# Branded Content Engagement Dashboard (Excel)

## Overview
This project analyses social media engagement data to demonstrate how branded
content performance can be monitored and reported using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers for interactive filtering
- KPI calculations (Total Engagements)

## Dataset Fields
- platform
- topic_category
- days
- likes, shares, comments

## Key KPI
- Total Engagements = likes + shares + comments

## Dashboard Preview
![Excel Dashboard](screenshots/dashboard social.png)

## Key Insights
- Engagement varies significantly by platform
- Certain topic categories consistently outperform others
- Engagement trends change over time

## Recommendations
- Prioritise high-performing platforms
- Focus on strong topic categories
- Monitor trends to refresh content strategy
